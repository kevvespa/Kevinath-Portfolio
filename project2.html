    • Banking App


    The Banking App is a secure banking application enabling staff to manage customer accounts and perform deposit and withdrawal transactions.

    Project Aim and Goal

    To develop a Banking Service Application to handle all the primary information required to maintain the customer account in a bank. Once the system is opened, the staffs have to pass through a login page to enter the main screen. Upon login, the system should display the staff’s name on the system’s user interface. Users are allowed to make transaction for deposit and withdrawal of the money.


    Stack and Explanation

    Stack: Java, Apache NetBeans IDE, Object-Oriented Programming (OOP), Java File Handling (text files).
    Explanation: The Banking Service Application was developed using Java and Apache NetBeans IDE, applying core and advanced OOP principles. It features staff authentication, customer registration, account management for savings and current accounts, and secure deposit/withdrawal transactions with balance validation. User interactions are handled through custom-designed GUIs, and all customer data is persistently stored using Java text file handling. The application ensures data integrity with thorough validation and error handling, incorporating concepts like classes, inheritance, polymorphism, interfaces, packages, and exception management.
