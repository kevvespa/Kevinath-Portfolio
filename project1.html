    • Food Recipe System (App)

    The food recipe system is a flexible and easy-to-use react native application that aims to make sharing recipes with friends, family, and co-workers easier.

    Project Aim and Goal

    To develop a Food Recipe/Cooking Guidance Application System, designed to cater to users of all levels, from students to housewives. A user-friendly platform offers a diverse database filled with healthy recipes, ranging from simple to advanced. Food tabs provides a list of ingredients and step-by-step guides. For enhanced learning, incorporated instructional videos demonstrating the exact cooking methods. For user engagement, offering options to suggest new additions to continually enrich the system.


    Stack and Explanation

    For this project, React Native was chosen as the programming language for its cross-platform support and performance advantages. Visual Studio Code was selected as the integrated development environment (IDE) due to its user-friendly interface, cross-platform compatibility, and alignment with React Native. Additionally, Firebase was deemed the most suitable and cost-effective database management system, offering a broader range of features compared to example MongoDB's free usage. Lastly, Windows 11 Pro was chosen as the operating system for its modern features and compatibility with the selected software tools. Overall, the chosen tools and technologies are well-suited for the project's development and implementation.
